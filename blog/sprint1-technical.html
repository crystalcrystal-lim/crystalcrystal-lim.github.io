CSS Padding, Border, and Margin

The CSS box model is the box that every HTML element is contained in. Every box is composed of four parts: Content, padding, border and margin. Understanding what these properties are and how to use them are important for creating desired layouts.

Content

The content is the text/image/video. In the example below I have two elements. One contains the text 'Content' which will be modified with padding, border and margin, and the other with the text 'Same size as the content box' that will remain the same to allow observations of the changes that will be made.

Padding

Padding clears a transparent area around the content. Length (px, pt etc.) percentage values or inherited values from a parent element are used to specify the property. In the example below 90px padding has been added. Padding is included in the total width of the element, and the difference can be observed below with the ‘content’ box in comparison to the second box.

Padding properties can be further specified to sides of the element using padding-top, padding-left, padding-bottom, padding right. The content box below has been modified to display 90px padding-bottom.

Border

Border goes around the padding and content area. It specifies the border color, style, and width using the properties border-color, border-style and border width. Borders are also included in the overall size of the element.

Margin

Margin clears a transparent area around the border, it is used to seperate from neighbouring elements. The values for this property are length (px, pt, cm, etc.), percentage, auto or inherited from a parent element.

Margin sides can be specified with margin-top, margin-left, margin-bottom, and margin-right. The example below has a 50px margin at the bottom. It is important to note that if the second box had a margin value as well, the total margin value will not be the two values added. The larger value will determine the size of the margin. Margins are not included in the total size of the element.
